# PJT08 - 은행 검색 페이지

## 프로젝트 개요
Kakao Map API를 활용하여 사용자 위치 기반의 은행 검색 기능을 제공하는 웹 페이지입니다.

## 개발 기간
- 2025.11.21

## 기술 스택
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Kakao Map API
- Kakao Mobility API (심화 기능)

## 주요 기능

### 필수 기능
1. **Kakao 지도 API 등록 (F01)**
   - Kakao Developers에서 JavaScript Key 발급
   - Live Server 플랫폼 등록 (http://127.0.0.1:5500/)
   - API 키를 apikey.js 파일에 저장 및 .gitignore 처리

2. **지도 출력 기능 (F02)**
   - Kakao Map API를 활용한 지도 표시
   - 초기 중심 좌표: 강남역 (위도 37.49818, 경도 127.027386)

3. **은행 검색 UI 구성 (F03)**
   - 광역시/도, 시/군/구, 은행명 3단계 드롭다운 구현
   - data.js 파일의 지역 및 은행 데이터 활용
   - 선택에 따른 동적 UI 갱신

4. **은행 검색 및 마커 표시 (F04)**
   - 사용자 선택 조건에 따른 은행 검색
   - 검색 결과를 지도에 마커로 표시
   - 마커 클릭 시 인포윈도우 표시 (은행명, 주소)
   - 새 검색 시 기존 마커 자동 제거

## 설치 및 실행 방법

### 1. API 키 발급
1. [Kakao Developers](https://developers.kakao.com/)에 접속 및 로그인
2. 애플리케이션 생성
3. 플랫폼 설정 → Web 플랫폼 추가 → `http://127.0.0.1:5500` 등록
4. 앱 키 → JavaScript 키 복사

### 2. API 키 설정
`js/apikey.js` 파일을 열어 발급받은 API 키를 입력합니다:
```javascript
const config = {
    kakaoMapApiKey: "발급받은_JavaScript_키",
};
```

### 3. Live Server 실행
1. VSCode에서 Live Server 확장 프로그램 설치
2. `index.html` 파일에서 우클릭 → "Open with Live Server" 선택
3. 브라우저가 자동으로 열리며 애플리케이션 실행

## 사용 방법
1. **광역시/도 선택**: 검색하고자 하는 지역의 광역시/도를 선택합니다.
2. **시/군/구 선택**: 광역시/도 선택 후 활성화되는 시/군/구를 선택합니다.
3. **은행 선택**: 검색하고자 하는 은행을 선택합니다.
4. **찾기 버튼 클릭**: 선택한 조건에 맞는 은행 지점이 지도에 마커로 표시됩니다.
5. **마커 클릭**: 은행명과 주소 정보가 인포윈도우로 표시되며, 심화 기능이 활성화된 경우 경로가 표시됩니다.

## 구현 과정

### 1. 지도 초기화
- Kakao Map API를 로드하고 지도 객체를 생성
- 강남역을 중심으로 초기 지도 설정

### 2. 검색 UI 구현
- data.js의 지역 및 은행 데이터를 활용하여 드롭다운 생성
- 광역시/도 선택 시 해당 지역의 시/군/구 목록 동적 로드
- 시/군/구 선택 시 은행 선택 박스 활성화

### 3. 은행 검색 및 마커 표시
- Kakao Maps의 Places 서비스를 활용한 키워드 검색
- 검색 결과를 마커로 지도에 표시
- 마커 클릭 이벤트 처리 및 인포윈도우 표시
